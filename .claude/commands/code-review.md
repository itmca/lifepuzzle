현재 브랜치의 변경사항을 코드 리뷰합니다.

## 리뷰 범위

```bash
git diff main...HEAD
```

## 리뷰 항목

### 1. 코드 품질
- 가독성, 중복 코드, 네이밍

### 2. 로직
- 엣지 케이스, 에러 핸들링, null 체크

### 3. 보안
- 민감 정보 노출, 입력값 검증

### 4. 성능
- 불필요한 연산, N+1 쿼리

## 출력 형식

```markdown
## 코드 리뷰 결과

### 요약
- 전체 평가: Good/Needs Improvement/Critical
- 변경 파일: N개

### 상세

#### [파일명]
**[Critical/Warning/Suggestion]** 라인 N
- 문제: ...
- 제안: ...

### 권장 조치
1. ...
```

## 심각도

| 심각도 | 기준 | 조치 |
|--------|------|------|
| Critical | 버그, 보안 | 반드시 수정 |
| Warning | 잠재적 문제 | 수정 권장 |
| Suggestion | 개선 가능 | 선택적 |
